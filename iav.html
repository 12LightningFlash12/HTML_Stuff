Images, Audio, and Videos


Page discription:

This page covers what can be done with images, audio, and videos.



Images

In HTML, you can add images in two ways:

  1. Use a link to link the images from an external URL
  2. Add the image in the folder the website is run from, then link it using the image's name and file extension

The things you can do with images, is:

  1. Chage the width
  2. change the height
  3. Add a border
  4. Round the corners
  5. Use an image as a link
  6. Floast the image
  7. Use the image as the page's icon
  8. Image maping

To find more information about images in HTML, follow the links below:

  • HTML Image Tag
  • HTML Images



Audio

Audio files are different, because they have to be in the website's folder. Also, what you can do with audio files spans to this:

  1. Add audio controls
  2. DOM events

Acceptable audio formats:

  • MP3
  • OGG
  • WAV

To find more information about adding audio to webpages, follow the links below:

  • Audio/Video DOM Events
  • HTML Audio
  • HTML Audio Tag
  • Audio Src Attribute

**Note: If the web browser does not support the audio tag, any text that is put inside the tag will be displayed instead.





Videos

You can add videos to a webpage in two ways:

  1. Having the video in the same file as the website
  2. Using a YouTube URL ina an iframe

Things you can do with videos in a website:

  1. Add video controls (pause\play, volume, download, and fullscreen toggle)
  2. Use autoplay
  3. Change the width and height
  4. Use JAvaScript to control the attributes of the video
  5. Use DOM events

TO find more information about adding videos to a website, follow the links below:

  • Video/Audio DOM Events
  • HTML Videos
  • HTML Video Tag

**Note: Before HTML 5, a video required a plug-in, Flash Player, to play. Now, with HTML 5, the video tag is used. Also, if the web browser does not support the video tag, any text that is put inside the tag will be displayed instead.
