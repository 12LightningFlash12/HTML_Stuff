Cascading Style Sheets


Introduction:

You may have gone to different websites, that had many different colors, images, or other visualy pleasing aspects; that is due to a web programming language, called Cascading Style Sheets (CSS). You may have also noticed different colors or borders within different pages of this website, that is also due to CSS.



Explanation


1. What CSS Does

Cascading Style Sheets, or CSS, is a web programming language, that formats how a website looks. CSS formats a website in many ways, for example:

  • Font color, family, size, and weight (fig. 1.1.1)
  • Deoration of text (fig. 1.1.2)
  • Background color\image (fig. 1.1.3)
  • Resizing images (fig. 1.1.4)
  • Borders on images and divs (fig. 1.1.5)


2. Adding CSS

2.1 Style Tag

The first way to add CSS into a website, is to use the <style> </style> tag. (fig. 2.1.1) shows how to add CSS using the style tag.


2.2 Link Tag

The second way to add CSS into a website, is to use the <link/> tag. (fig. 2.1.1) shows how to add CSS using the link tag.



3. CSS Selsctors

3.1 Classes

The class selector matches an element with the value of their class attribute. The value of the class attribute must be the same as its corresponding selector. Also, more than one HTML element can have the same class value. (fig. 3.1.1 & fig. 3.1.2) shows an example of the class selector syntax.


3.2 IDs

The ID selector matches an element based off the value of their id attribute. The value of the id attribute must be the same as its corresponding selector. Also, more than one element can have the same id value. (fig. 3.2.1 & fig. 3.2.2) shows an example of the id selector syntax.



4. CSS Box Model

First, in HTML, all elements are considered as a box. 'Box model' is a term used when looking at design or layout. Box Model wraps around all HTML elements. At the core, is the content (i.e. text or images), then you have padding, borders, and on the outside, margins. (fig. 4.1.1) shows an example of the Box Model.
