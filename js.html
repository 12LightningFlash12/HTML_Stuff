Basics:

First let's understand what JavaScript does within webpages; JavaScript conrols the website's behavior. This covers things like buttons, DOM or Document Object Model Events, changing HTML content depending on some form of input, etc. Basically, if it makes the webpage behave in a certain way, that is done using JavaScript.

2. JavaScript HTML DOM Events

2.1 What DOM Is

DOM Events, or Document Object Model, allows event-driven programming languages, such as JavaScript, to register variuos events. (fig. 2.1) shows the DOM Tree of Objects. With the DOM Tree, JavaScript has enough power to create a dynamic webpage. With this power, JavaScript can do the following:

  • Change the HTML elements
  • Change the HTML attributes
  • Change the CSS styles
  • Remove HTML elements\attributes
  • Add HTML elements\attributrs
  • React to HTML events
  • Create new HTML events
